Features

•Gift Aid management for the United Kingdom.

 

SARS 2023 s18A field requirements

The following new fields required by SARS are included in the base s18A receipt template available in the system:

oDonor name / Legal name of entity

oTax number

oID number / Passport number / Organisation registration number

oCell number

oPhone number

oEmail address

oAddress

To begin using this template, simply clone it and apply your branding.

 

How to use the sample template on DevMan

SARS has published a sample layout for the s18A receipt.

•We have added a SARS-compliant template that can be customised.

•Find the sample template by going to Finance > 18A receipts > Setting > Templates > Print.

•Click on the template to open, Save a copy, then add your own branding and organisation details.
