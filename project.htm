Overview

 

Use this feature to track and report on applications for funding and approved projects.

Manage project workflow, financials and documentation in one place.

 

Topics

 

Features

•Visualise your data via the dynamic, interactive and configurable dashboard

•Add links to people, organisations and assign to project managers

•Link one or more projects to an organisation

•Track the progress of your projects using a life cycle or milestone

•Add links to related communication and attach important documents

•Create unique groups of data with customised reports

•Set up an online grant application to streamline funding requests
