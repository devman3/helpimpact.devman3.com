Overview

 

Employ your branded online results page to gather, track and monitor beneficiary submissions.

 

Topics

 

Features

•Bulk generate required semesters for the cohort

•Bulk send results requests

•Collect individual subject outcomes and an electronic copy of the transcript

•Manage results collection workflow for each beneficiary - from Requested to Captured to Verified

•Effortlessly follow up on outstanding submissions

 

Steps

There are a few important steps required in order to send a request for results to be submitted online:

1.Use Bursary > Search Advanced to filter for the group of bursaries for whom results are required.

2.Bulk generate the semester from the search results.

3.Send the online results request via the Academic results grid.  This can be accessed via:

oBursary > Academic results grid  - use this option if there are no application results on DevMan for the current year.

oPeople > Search Advanced - use this option if there is a mix of applicant and beneficiary results for the current year.

4.Monitor and verify the incoming results.

 

Important reminders

•Results links must be sent from View academic results grid.

•Auto generated links are unique and specific to the student and semester

•Semester status must be Requested

•When results are submitted, the status will change to Captured

•Once updated to Captured, the online page will lock
