Manage the projects in your online giving project list

Add a project to the list. 

Arrange the list in alphabetical order. 

Remove a project from the list. 

Under the Finance menu, go to Integrations then Donation pages.

Click through on the online giving page that you want to update.

Go to Projects and select Add new project

Enter the project name and select. 

Click on Default and the project name will autofill under Description. 

The project has been added to the list. 

To arrange the list alphabetically, simply click and drag the project to where it should be. 

To remove a project from the list, click on the Edit button and Remove. 

Save the new project list. 

Click on Preview  to see the list your donors will see. 

The project list is up to date.
