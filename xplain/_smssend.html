Send bulk SMS

Follow these three steps:

  • Add a template for your SMS message. 

  • Add a mailshot and save. 

  • Add the recipients to the mailshot using Search advanced and Send.

In this example, we will send a bulk SMS to current bursary students. 

Go to Communication > Mailshots > Templates > SMS and add your message. 

Add a name for your batch of SMS messages.

Add your message using any merge tags you wish to use and save.

Go back to Communication and add a mailshot.

Add a name for your mailshot.

Select a subscription and then add Recipients using Search advanced.

Early registration

Add the results to the mailshot.

The selected recipients have been added successfully.

Use the breadcrumb to go back to the mailshot page.

To send the SMS go to Recipients > Send communication > SMS.

Click on Yes and proceed.

Select the prepared template from the drop-down list then Send.

The batch of SMS will be sent.
