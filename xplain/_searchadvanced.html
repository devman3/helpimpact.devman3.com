Search advanced and saved searches

Here is an example showing how to add different filters to search across multiple fields.

The filters you create can be saved for re-use and shared with colleagues.

Search advanced is conveniently available where needed.

Build reports, spreadsheets and dashboards with criteria from a wide variety of data sources.

We are looking for projects that are:

  • managed by a particular manager,
  • and who have Approved status. 

Begin by adding selection criteria for the Project.

Select the Project officer from the drop-down list. 

Select Workflow phase.

Click on Add to results.

173 projects meet the criteria.

We can now add further criteria.

We want to find which if these projects received payments in a current period.

Click on Add selection criteria and then select Payment.

Enter the dates and click on Add to results. 

5 projects received payments and all five meet all the criteria.

To save the search for future use, give it a name and save. 

To enable all to use this search, click on Make public.

Select view to see the results on the screen.

You have now seen how to use a few of the many features of the Search advanced function.

Don't forget that this is available in all areas of DevMan.
