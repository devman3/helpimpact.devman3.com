Add a new user

Add a person profile using their full name and email address.

Click on affiliation and tick For internal use.

TIP: For a detailed explanation of security roles please go to advanced security.

In this example, we will add a user whose role is a finance administrator.

Use inline search and link the person profile.

Add login name.

Select prompt.

Add a password and confirm.

TIP: Please advise user to change the password once they've logged in.

Select Add.

Go to Permissions and select the appropriate role from the drop-down list.

If the user will send email from DevMan, select Add default email identity.

To verify the email address, go to Identities and click on the notepad icon.

Click on Check to see if the email address has been verified.

If it has not been verified, click on Request and an email address will be sent to the user enabling them to verify the address.

NOTE: Remember to return to this page on the User profile to see if the verification has worked - click on Check.

The new user has been added.

Select the user here and send a Welcome email which will include the login details, username and password.
