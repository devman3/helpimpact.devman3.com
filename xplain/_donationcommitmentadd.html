Add a new commitment

In this example, a commitment is added from the Person profile.

Select Add > Financials > Add commitment > Standard.

This will default to the donor profile.

Select the Project and add a description.

If the donor wishes to remain anonymous, tick this box.

Enter the rest of the details to suit your needs.

Leave this unticked.

Select the frequency of the donations.

Enter the amount per donation.

Either enter the End date or the Number of donations.

Enter the Start date.

Select Save and close.

A new commitment has been added.
