Link projects to a meeting

See how to:

  • find and link a group of projects to a meeting 

  • add a single project to a meeting

Go to Development > Projects > Search. 

In this example we search for projects with current status Under consideration in a defined time period.

TIP: Search using any of these available options. 

Select workflow phase, consideration dates, and tick Current status only, then Search.

Select Add to a meeting.

Select the meeting from the drop-down list, then Begin process. 

The projects have been linked successfully.

Click through on Linked projects to view the individual projects.

If you need to add an additional project to the meeting, do so from the project profile page.

Select Add > Link to a meeting and proceed as before.

Select the meeting from the drop-down list, then Begin process. 

The additional project has been linked successfully.

The projects have been linked to the meeting.
