Search

Search is available in all areas of DevMan.

Find individual records, or all records that meet certain criteria.

Follow the same process in all areas and across multiple fields.

Watch the rest of this tutorial to see how to:

  • Search for a person using their ID number.
  • Search for organisations with two different focus areas.
  • Search for projects that support certain causes.

TIP: If you can't find a specific record, increase the number of grid rows on your preferences, or go to the Search page and do a broader search.

Enter the number you are searching for, then select ID number in the drop-down list. 

Your results will display on a grid like this one.

Select the Focus areas from the drop-down list, then select Search. 

TIP: To remove an area from the search, click on it.

Select the Cause from the drop-down list, then select Search. 

TIP: To remove an area from the search, click on it.

You have now seen how to use a few of the many features of the Search function.

Don't forget that this is available in all areas of DevMan.
