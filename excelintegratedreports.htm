Features

•Generate and save re-usable reports.

•View interactive dashboards for enhanced data analysis.

•Simply refresh data to reflect the latest information.

 

Using the EIR report

•Save a copy of the report on your network / PC.

•When opening the report, you may be prompted to enable content and make it a trusted document – accept these prompts.

•Dashboard – this sheet reflects high level statistics of applications based on the source data.

•Source data – the other sheets list details for the applications.  These sheets draw the information from DevMan and will show with alternating green and white rows.

•To refresh the report:

oGo into each source data spreadsheet, right click on the page and select ‘Refresh’. This will pull through any new data from DevMan.  

oGo to the Dashboard sheet, select the Excel Data tab, then Refresh all to update the pivot tables and graphs.

oIf you get an error message referring to pivot tables overlapping, add in additional rows between pivot tables.

How to create an EIR report

The steps to create an EIR are as follows:

•Create a saved search using search advanced.

•Select Export or Statistics.

•Create a saved Export or Statistics selection, and then select Publish report.

•Save the published report, and then select Extract > To link. Copy the link.

•In a new Excel spreadsheet, select the Data tab. Then select the From Web button, and paste this as a Basic URL.

•Select your saved report you want to import, and then select Transform data. (For Excel help follow the link provided in the pop-up)

How to create an EIR report compatible with Mac or Windows

In the first phase you will need to create an .iqy file. This is just the query with the link to DevMan.

You can create this by doing the following:

•After generating your published reports, in the list view of the report, click Extract > To link.

•Copy the link.

•Open Word or a Text editor.

•Paste the DevMan published reports link into the document.

•Click Save as and name your file. (Save to a place where you will be able to locate it)

•On the document file itself, double-click the name so that you highlight it – as if you were wanting to change the name.

•Delete the file extension only, and replace it with .iqy

•Example: If you used Word to create the file, it will say “exampledata.docx”. Change that to “exampledata.iqy”.

 

In Windows Excel - compatible with Mac

The second phase will take place in Excel.

In Excel, go to Data tab > Existing Connections.

•In the pop-up, click the Browse for More button.

•Navigate to the folder where you saved your .iqy file. Select it and click Open.

•On the Import Data pop-up, select whether you would like to add the data to the current tab selected, or to a new tab and click OK.

Data should now be loading into the cells. When you click refresh it should update the data, whether you are on Mac or Windows.

To refresh your data, follow these steps:

•Click the Data tab at the top.

•Click the Refresh All button in the toolbar.

 

In Mac Excel - compatible with Windows

The second phase will take place in Excel on your Apple device.

•With Excel open, click the Data > Get External Data > Run Web Query menu item in the Mac navigation top ribbon (not the Excel navigation).

•Locate your .iqy file and click OK.

Data should now be loading into the cells. When you click refresh it should update the data.

Please note that creating an EIR in this way will allow for compatibility on both Mac and Windows devices.

To refresh your data, follow these steps:

•Click the Data tab at the top.

•Click the Refresh All button in the toolbar.
