Overview

 

Cut the administrative burden, take advantage of our online application tool.

Allow your bursary and grants applicants to apply online, and integrate these forms into your organization workflow.

 

Topics

 

Features

•Pre-screen and auto reject non-compliant applications.

•All applications and documentation are submitted directly into DevMan.

•Initiate timely submissions with automated workflow reminders.

•Bulk close out incomplete applications when the period is closed.

•Easily report on the results of the application cycle.

 

Getting started

•Please notify the DevMan team of when the application should open and close.

•It is important to bulk update all incomplete applications to closed (Declined - Incomplete).

•It is optional to send a decline notification if applications were not completed.

Extract a link from a specific application

If you need a token number for a specific applicant and/or phase, follow these steps:

•Navigate to the application.

•Scroll down to Recent touchpoints.

•Click on the notification that you need the token from.

•Click on the Source button on the message editor.

•Scroll down until you see a link that looks like the following example:

ohttps://company.devman.co.za/devman/external/?t=994c434d&amp;t1=566f0788

•Delete the “amp;” from the link. It should now look like this:

ohttps://company.devman.co.za/devman/external/?t=994c434d&t1=566f0788

You can now access this application yourself or send this link to others.
